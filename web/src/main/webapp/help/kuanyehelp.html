关注款爷赢家微信公众号，进入款爷盈家。

  • 1. 用户注册

    • 点击“收银台”、“推广”、“我的”都可以进入注册页面，填写个人信息(邀请码选填)。

  • 2. 如何绑卡

    • 方法1：进入“收银台”，填写持卡信息(请确保银行卡号务必正确)。方法2：进入“我的”，点击“我的银行卡”进入绑卡页面，填写持卡信息(请确保银行卡号务必正确)。

  • 3. 刷卡消费

    • 方法1：进入“收银台”，输入金额点击“确定”，生成二维码，可自己支付；分享二维码，供他人扫描方法2：进入“我的”，点击二维码图标进入付款码页面，二维码可面对面扫描，也可分享给他人扫描，输入金额，完成支付。

  • 4. 理财

    • 进入“我的”，点击“款爷理财”，进入理财页面，点击理财产品，进入投融家，自动获取手机号码，填写基本信息进行注册。

  • 5. 借款

    • 进入“我的”，点击“款爷借款”，进入借款页面，点击理财产品，输入金额，调整还款期数，点击“下一步”进入基本信息填写页面，填写信息，提交。
