 鉴于：甲方为有借款需求并在乙方平台上进行了注册，同意乙方《投融家注册协议》的借款申请人。

乙方是杭州投融谱华互联网金融服务有限公司，互联网理财服务平台“投融家”（网址为www.tourongjia.com） 的运营方。

甲、乙双方就乙方为甲方提供居间服务一事，依据《合同法》《民法通则》及相关法律、法规的规定，双方在自愿、平等的原则下经协商一致，签订如下协议：

一、释义
    在本合同中，除非上下文另有解释，下列词语具有以下含义：
    1. 借款申请人：指有借款需求，在乙方的平台进行注册，同意乙方《投融家注册协议》，并向乙方提供借款资料供其审核，并同意通过乙方平台发布借款信息的借款方。
    2. 借款合同：指经乙方居间而由甲方与出借人以合同的形式订立的、约定甲方向出借人借入一定数额的出借资金，甲方到期还本付息的借款合同。
    3. 借款人：指在乙方的平台注册，同意乙方《投融家注册协议》，并通过乙方审核的、符合乙方关于借款人资格规定的、与出借人签订《借款合同》的借款方。
    4. 出借人：指在乙方平台注册，并通过平台与借款人签订《借款合同》的出借方。

二、甲方的权利和义务
1、甲方应当按如下要求向乙方提供和更新甲方信息：
    （1）甲方应及时向乙方提供乙方要求的各项借款人基本信息，并保证及时更新甲方的个人信息，以确保诸如个人身份信息、电子邮件地址、联系电话（含手机号码）、联系地址、邮政编码等个人资料或企业名称、注册地址、法人代表、注册资金、经营范围、组织机构代码等信息和证明文件的有效性、安全性和及时更新，并应确保提供的信息真实、完整、准确。如已经提供的借款人基本信息发生任何变化，甲方应及时通知乙方，并应乙方要求提供相关证明文件。
    （2）如因甲方违反上述规定，提供的信息不真实、不完整、不准确、不及时和/或未及时在各类信息发生变更后及时通知乙方并提供证明文件，所造成的一切损失和责任由甲方承担。
2. 甲方应授权乙方收集、了解甲方的各项信用信息，包括但不限于其在中国人民银行的征信报告、工作情况、收入情况、家庭成员情况、使用投融家平台的情况、历史债务清偿情况以及其他各类信息等。
3. 甲方有权按照约定通过乙方平台就自己的资金需求发布信息，查阅交易机会，签订和查阅合同，进行资金充值、提现。
4. 甲方应按约定向乙方支付居间报酬费用。

三、乙方的权利和义务
1. 借款人信息的提供和收集

    （1） 乙方有权要求甲方向其提供各类借款人基本信息，包括但不限于甲方的姓名、身份证明及证件号码、住址等个人基本信息，以及账户信息、工作情况、收入和资产情况、家庭成员情况以及个人信用信息及相关证明文件等；或要求甲方提供企业名称、注册地址、法人代表、注册资金、经营范围、组织机构代码等信息和证明文件。
    （2） 如因甲方违反上述规定，提供的信息不真实、不完整、不准确、不及时和/或未及时在各类信息发生变更后及时通知乙方并提供证明文件，所造成的一切损失和责任由甲方承担，乙方同时有权立即停止为甲方提供服务并视情况限制或禁止甲方使用投融家平台的全部或部分功能。
    （3） 经甲方授权，乙方可以自行或通过乙方合作机构自公开及私人资料来源收集甲方的额外资料和信息，搜集的资料和信息范围包括但不限于：
        ①甲方的身份信息，包括但不限于姓名/名称、证件号码、证件类型、住所地、电话号码以及其他身份信息；
        ②甲方在乙方的关联方和合作伙伴处留存以及形成的任何数据和信息，包括但不限于甲方的证券账户和/或银行账户信息、甲方在乙方关联方或乙方合作伙伴的网络平台上留存、形成、产生的有关法律文件以及任何个人信息或行为数据；
         ③甲方的信用信息，包括但不限于甲方征信记录和信用报告；
        ④甲方的财产信息，包括但不限于甲方的店铺/企业经营状况、财税信息、房产信息、车辆信息、基金、保险、股票、信托、债券等投资理财信息和负债信息等；
        ⑤行政机关、司法机关留存的任何信息，包括但不限于甲方的户籍信息、工商信息、诉讼信息、执行信息和违法犯罪信息等； 2. 借款人信息的使用和披露
甲方同意乙方有权按照如下规则使用及公开其各项借款人信息：

    （1） 在投融家平台公示全部或部分借款人信息。
    （2） 乙方可在业务运营中向其第三方合作机构披露全部或部分借款人信息，但乙方与该第三方合作机构签署的合作协议中应对第三方合作机构保护借款人信息作出规定。
    （3） 乙方可在业务运营中进行数据分析、行业研究、市场推广等情形下整体使用借款人信息。
    （4） 在行政、司法等政府机构要求时，乙方可向该等政府机构提供借款人信息。
    （5）当甲方违反本协议、各交易服务规则和/或甲方通过投融家平台签署的法律文件（包括借款交易协议）之约定或任何法律法规规定时，乙方有权根据自己的判断和/或有关法律文件的约定，披露乙方的个人信息和资料以及甲方的违法、违约行为和对应的交易文件而无需承担任何责任，披露的方式包括但不限于：①在投融家平台及其他网站公布；②向与甲方交易的主体以及有权的仲裁机构和司法、行政等权力机关提供；③向受乙方或与甲方交易之其他投融家平台用户委托的专业法律顾问、催收机构等服务机构提供；④其他披露方式。
    （6）乙方有义务对甲方的个人信息和资料（包括甲方提供的和乙方搜集、记录的所有数据、信息、资料和相应的交易行为记录、交易文件）承担保密义务，不将乙方信息向第三方进行出租或出售。
3、撮合服务
    乙方根据其获取的甲方信息及自身的商业判断决定为甲方资金借入提供居间服务的，将在乙方成立并运营的有投融家平台上对甲方及其资金需求进行展示，并使用投融家交易平台撮合甲方与一个或多个出借人达成借贷交易、签署借款协议。
4、资金划转托管服务
    乙方在其合作银行（“监管银行”）以乙方名义开立的、资金独立于乙方自有资金的银行监管账户（“监管账户”），并通过监管账户，根据出借人与甲方的资金划转指令和委托，委托监管银行、第三方支付机构等协助完成出借人与甲方之间借款协议项下资金的划转（包括出借人出借给甲方和甲方向出借人还款）。
5、借款人信用审核
    乙方有权根据甲方通过投融家平台提供的信息以及乙方根据甲方的授权自行搜集的信息进行审核并针对甲方提供的信息对甲方的还款能力、还款意愿和资产状况进行评估、分析，基于此等评估、分析，确定、更新甲方可以通过投融家平台的推荐、撮合服务获得的借款额度。
6、借贷交易管理
    在甲方于出借方达成借贷交易后，乙方将对甲方与出借人之间的借贷交易提供持续的管理，包括但不限于还款提醒、催收、信用跟踪评审等。

2. 双方同意，当出借人通过在投融家平台对甲方的借款要约进行点击投资时，甲方与出借人订立的借款协议成立。在甲方的某一项借款要约全部得到满足而相应的所有借款协议生效后，出借人授权乙方通过委托第三方支付机构及监管银行划扣其在监管账户中出借人个人虚拟账户项下资金的方式向甲方指定的以下账户划转其借入的资金，甲方须确保如下账户为甲方名下合法有效的银行账户，甲方变更该账户时必须以传真及电子邮件方式向乙方发出通知、并经乙方确认后方可变更；如因甲方未及时书面通知乙方而引发的损失由甲方自行承担：

3. 甲方在借入资金后应按照相应借款协议之约定按期还本付息。甲方应按照借款协议约定的每月还款日通过第三方支付账户向其投融平台账户充值的形式支付其当期应支付的本息之和。乙方将根据其与出借人之间的约定将甲方账户中的该等资金自动划转至出借人的投融家平台账户。
4. 若在甲方通过投融家平台公示的规定时间内，甲方的某一项借款要约未能全部得到满足或乙方根据本协议相应规则撤销了甲方的该项借款要约，则已点击进行投资的出借人与甲方订立的借款协议自动撤销。
5. 甲方未在到期日前向其投融家账户按照相应借款协议之约定充值以缴付当期本息的，视为甲方违反借款协议及本合同的约定：
    （1）甲方除应尽快按期偿还借款本息外，还应按照借款协议的约定的方式缴付相应的逾期罚息、滞纳金等；
    （2）乙方还有权采取以下措施督促甲方尽快履行借款协议，包括但不限于向出借人进一步提供甲方的信息以供出借人催收、通过第三方合作机构催收借款等。

五、 服务费
1. 就乙方向甲方提供的本协议项下服务，甲方应向乙方支付居间服务费。服务费将按照甲方借入资金的本金总额收取，具体计算方式见协议附件一（收费确认单）。
2. 上述公式计算出的服务费应由乙方在甲方向出借人借款成功时一次性从向甲方支付的本金中扣除。
3. 此外，为了所有出借人的共同利益考虑，乙方以乙方名义单独开设并管理“投融家借款人风险保证金账户”。当甲方借款成功后，乙方将按照借款产品类别及乙方给甲方评定的信用等级等信息计提风险保证金。计提的风险保证金将存放入“风险保证金账户”进行专款专用管理并在借款成功时一次性从向甲方支付的本金中扣除。乙方将在投融家平台公示风险保证金的计提比例。

六、限制责任

1. 非因乙方的原因导致的甲方在使用乙方服务的过程中遭受的损失（损失原因包括但不限于甲方的操作失误导致的损失、第三方机构导致的损失、交易对手违约导致的损失，甲方使用的电脑、手机终端设备感染病毒或安全防御系统被突破造成甲方无法正常使用投融家平台服务或账户、个人信息泄露而导致的损失，损失类型包括但不限于本息损失），均应由甲方自行承担或解决。
2. 乙方不绝对保证甲方的借款需求能够得到满足，也不就甲方的借款需求未得到满足而造成的任何损失承担任何责任。

七、违约责任
任何一方违反本协议的约定，使得本协议的全部或部分不能履行，均应承担违约责任，并赔偿对方因此遭受的损失（包括由此产生的诉讼费和律师费）；如双方违约，根据实际情况各自承担相应的责任。违约方应赔偿因其违约而给守约方造成的损失，包括合同履行后可以获得的利益，但不得超过违反合同一方订立合同时可以预见或应当预见的因违反合同可能造成的损失。

八、争议的处理
本协议在履行过程中，如发生任何争执或纠纷，双方应友好协商解决，协商不成的，任何一方有权向 乙方 人民法院提起诉讼。
